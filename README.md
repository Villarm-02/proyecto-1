# SIMULADOR DE DECISIONES PARA PLATAFORMA DE STREAMING
Este proyecto consiste en el desarrollo de un sistema en C# que simula la consola de decisiones de una plataforma de streaming. Su objetivo es evaluar contenidos enviados (películas, series, documentales o eventos en vivo) y determinar si pueden publicarse según reglas técnicas y de impacto.

El programa analiza cada solicitud en tiempo real y genera una decisión basada en criterios como duración, clasificación, horario, tipo de contenido y nivel de producción.
OBJETIVOS:
Demostrar el uso de estructuras de control y lógica de programación para resolver un problema real mediante un sistema interactivo.
